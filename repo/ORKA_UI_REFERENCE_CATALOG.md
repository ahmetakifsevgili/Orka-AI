# Orka UI Reference Repository Catalog

This folder is a local inspiration/reference library for Orka frontend redesign work.
The repos are not part of Orka runtime and should not be copied blindly. Check each
project's license and adapt only the patterns/components that fit Orka's product.

## Orka Design Direction

For Orka, the strongest path is not a copied template. Use this reference stack:

1. Landing architecture: `launch-ui`, `mainline-nextjs-template`, `cruip-tailwind-landing`.
2. Premium motion: `magicui`, `uilayouts`, `motion-primitives`, `cult-ui`.
3. App shell/product quality: `dub`, `midday`, `twenty`.
4. Tutor/chat workspace: `lobe-chat`.
5. Component details: `originui-shadcn`, `tailgrids`, `eldoraui`, `animata`.

The Orka landing should be a sticky-scroll product story:

Mission Control -> Tutor Repair -> Study Room -> Review -> Exam War Room ->
Source/Wiki Pro -> Notebook Studio -> Code IDE -> Progress/Memory.

The logged-in app should stay operational, dense, calm, and scannable rather than
marketing-heavy.
