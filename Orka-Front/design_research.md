# Design Research Notes

## Linear.app
- Dark background with very subtle hierarchy (zinc-950 sidebar, zinc-900 main)
- Clean typography with Inter font
- Sidebar with collapsible sections, very clean icons
- No flashy colors - monochrome with green accent only for status
- Subtle borders (zinc-800 level) for separation
- 3-pane layout: sidebar + main content + detail panel
- Animations are minimal and functional
- Premium feel through spacing and typography hierarchy
- Hero section: large bold typography, no images, product demo below
- Landing page uses product screenshots as hero, not illustrations

## Raycast.com
- Command center feel - input-focused UI
- Dark background with bold hero typography
- Abstract 3D visual in hero section
- Clean sections with product demos
- Keyboard-first design philosophy
- Social proof with testimonials
- Feature sections with interactive demos
- Very clean footer with newsletter

## Key Design Patterns for Orka AI Landing Page:
1. Hero: Large bold typography + abstract visual/product demo
2. Feature sections: Show actual product UI, not illustrations
3. Social proof section
4. Clean CTA buttons
5. Monochrome dark palette with subtle depth
6. Typography-driven hierarchy
7. Smooth scroll animations between sections
